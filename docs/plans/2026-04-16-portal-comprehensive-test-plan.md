## 第三輪：邊界與壓力測試 (Edge Case Test)

| ID | 項目 | 操作 | 預期 |
|----|------|------|------|
| E-01 | 空列表 | portal 存取無資料模組 | 空狀態提示 |
| E-02 | XSS 注入 | 搜尋 `<script>alert(1)</script>` | 不執行、正確轉義 |
| E-03 | 超長輸入 | 搜尋 500+ 字元 | 不崩潰 |
| E-04 | 快速連點 | 連續點擊按鈕 | 不重複觸發 |
| E-05 | 無效 URL | `/my/orders/99999` | 404 或錯誤頁 |
| E-06 | 未登入存取 | 清 cookie 後 `/my/home` | 導向登入 |
| E-07 | Portal 越權 | portal 訪問 `/odoo/crm` | 被拒絕 |
| E-08 | Console 錯誤 | 全程監控 | 無 error |
| E-09 | 手機橫屏 | 切換 844×390 | 排版不崩潰 |
| E-10 | 分頁邊界 | 超出範圍頁碼 | 友善處理 |
| E-11 | 密碼錯誤 | 錯誤密碼登入 | 錯誤訊息 |
| E-12 | 中文搜尋 | 搜尋「專案」| 正確過濾 |

## 執行策略

- 每輪獨立執行，產出 PASS/FAIL 結果
- 截圖存入 `/tmp/screenshots/test_round{N}/`
- Console 錯誤收集
- 最終產出 markdown 測試報告
